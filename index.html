FX REST API

❓ Getting Started ⌄
  1. Daftar dan ambil API Key dari menu Keys.
  2. Kirim request dengan header x-api-key.
  3. Pilih pair forex, mis. EURUSD atau USDIDR.
  4. Cek sisa credit lewat endpoint /account/usage.
